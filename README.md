# horse-opendata

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A project that analyzes a dataset of four-character idioms and their associated animals.

## Features
- Checks a CSV file of idioms for any entries where the idiom does not contain the associated animal
- Logs any discrepancies to the console

## Data
The project uses a CSV file named `idiom4.csv` that contains the following fields:
- `四字熟語` (Four-character idiom)
- `読み` (Reading)
- `簡単な意味` (Simple meaning)
- `動物` (Animal)
